# Malaria Reporting System

Web-based malaria case reporting dashboard built with React, Vite, Tailwind CSS, and Supabase.

## Local development

Requirements:

- Node.js
- npm

Install dependencies and start the app:

```sh
npm i
npm run dev
```

## Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase
